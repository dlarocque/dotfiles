# dotfiles

Quick and easy dotfiles, managed by GNU Stow

## Prerequisites
- GNU Stow >= 2.3.1
- Neovim >= 0.6.1


## Install

Clone dotfiles in `.dotfiles` in the home directory.
```zsh
$ git clone https://github.com/dlarocque/dotfiles ~/.dotfiles
```

Install Neovim config
```zsh
$ stow nvim
```

Install zsh config
```zsh
$ stow zsh
```
